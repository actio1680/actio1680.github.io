Base de datos de las sentencias de la Corte Suprema de Justicia del Perú 1870-1972

El visor de sentencias del siglo XIX y XX permite la visualiziación demostrativa del archivo BD-AJCS-XX-XXI.CSV; por lo que, es una versión ligera, para visualizar el archivo completo descargar el archivo XLSX o CSV.

Para informarse sobre los detalles técnicos de la base de datos leer el artículo principal.
